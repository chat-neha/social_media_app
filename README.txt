A basic implementation of a social media app using MySQL and PHP connectivity with HTML. 

Pages include:
1. Login/Sign in page 

If you are a new user, you are directed to the 2. Users list page, where you can choose whom to follow.

An existing user will be lead to their 3. Homepage, where they can view their followers' published content. They can like and comment on posts. 

4. Explore page - Customised based on user's like history 
5. Message - to send messages to friends 
6. Post - To create their own post.
7. Search User - to look for users and reach out to them via messages. 

Executed on localhost using Xampp local server.
